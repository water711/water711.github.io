Python之excel文件追加内容

Feb 2, 2018
1 min read

首先要安装三个模块：xlrd，xlwt，xlutils

命令：pip install xlrd xlwt xlutils

 

示例代码：

#!/usr/bin/env python

-*- coding:utf-8 -*-

from xlrd import open_workbook from xlutils.copy import copy

#读取excel文件 r_xls = open_workbook(“test.xls”)

获取已有的行数

row = r_xls.sheets()[0].nrows

将xlrd的对象转化为xlwt的对象

excel = copy(r_xls)

获取要操作的sheet

table = excel.get_sheet(0)

#对excel表追加一行内容，括号内分别为行数、列数、内容 table.write(row, 0, ‘内容1’) table.write(row, 1, ‘内容2’) table.write(row, 2, ‘内容3’)

#保存并覆盖文件 excel.save(“test.xls”)
