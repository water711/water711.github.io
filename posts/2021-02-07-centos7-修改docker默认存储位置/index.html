Centos7 修改Docker默认存储位置

Feb 7, 2021
1 min read

 

1、查看docker默认目录

docker info | grep “Docker Root Dir”

2、新建或者编辑 daemon.json 文件

vim /etc/docker/daemon.json

{ “graph”: “/mnt/docker” }

3、重启docker服务

systemctl restart docker

4、查看docker目录文件

ls /mnt/docker
