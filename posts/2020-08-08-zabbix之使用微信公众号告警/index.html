三、编写Python脚本

#!/usr/local/python3/bin/python3

import requests import json import sys

def get_token(appid, secret): url = “https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=” + appid + “&secret=” + secret response = requests.get(url) ret = response.json() return ret[‘access_token’]

def send_message(token): url = “https://api.weixin.qq.com/cgi-bin/message/template/send?access_token=” + token data = { “touser”:to_user, “template_id”:template_id, “data”:{ “keyword1”:{ “value”:event_time, “color”:"#993399" }, “keyword2”: { “value”:host_name, “color”:"#0099CC" }, “keyword3”: { “value”:event_name, “color”:"#FF0000" } } } data = json.dumps(data) response = requests.post(url ,data=data)

appid = “wx11111111111111” #公众号appid secret = “fa582222222222222222222” ##公众号secret template_id = “3mYrEo_MB8rCsBoPBF5h80nsjYR2koGnZ” #公众号模板ID

to_user = sys.argv[1] # zabbix运行脚本时传入的第1个参数 message = sys.argv[2].split(’;’) # zabbix运行脚本时传入的第2个参数

event_time = message[0] host_name = message[1] event_name = message[2]

token = get_token(appid, secret) send_message(token)

将脚本移动到Zabbix指定脚本目录

[root@zabbix ]# cat /etc/zabbix/zabbix_server.conf | grep AlertScriptsPath #查看zabbix配置文件，找到存放脚本的目录

[root@zabbix ]# chmod +x alert_weixin.py [root@zabbix ]# mv alert_weixin.py /usr/lib/zabbix/alertscripts/

 

四、配置Zabbix

1、创建报警媒介类型：管理->报警媒介类型->创建媒体类型

2、为用户添加报警媒介：管理->用户->报警媒介->添加

类型：微信公众号（报警媒介类型）

收件人：公众号用户oppenid（对应{ALERT.SENDTO}变量，将传入脚本第1个参数）
