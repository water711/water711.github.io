pthon3实现oracle数据库查询、导出EXCEL并邮件发送

Posted by on Thu, Jan 2, 2020

一、准备环境

1、安装Oracle 客户端（python、oracle客户端的位数要一致）

2、安装Python第三方模块

pip3 install cx_Oracle xlsxwriter

cx_Oracle模块： 一个允许访问Oracle数据库并符合Python数据库API规范的模块。该模块目前针对Oracle客户端19c、18c、12c和11.2以及Python2.7、3.5、3.6、3.7和3.8进行了测试。

xlsxwriter模块： 编写Excel 2007+XLSX文件格式文件，可用于将文本、数字和公式写入多个工作表，并支持格式、图像、图表、页面设置、自动筛选、条件格式和许多其他功能。

二、编写代码

# -*- coding: utf-8 -*-

import time
import smtplib
import cx_Oracle
import xlsxwriter
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart
from email.mime.application import MIMEApplication

def send_mail():
    from_addr = "12345678@qq.com"  #发件人
    pwd = "zjjeiuwpaynojhgsdf"     #授权码/密码
    to_addr = "87654321@qq.com"    #收件人

    msg = MIMEMultipart()    #实例化对象
    msg["Subject"] = "销售报表"    #主题
    msg["From"] = from_addr  
    msg["To"] = to_addr

    part1 = MIMEText("这里是邮件内容")
    msg.attach(part1)
 
    # 添加附件
    part2 = MIMEApplication(open(date + ".xlsx", 'rb').read())
    part2.add_header('Content-Disposition', 'attachment', filename=date + ".xlsx")
    msg.attach(part2)
 
    # 连接smtp服务器，登录服务器并发送文本
    smtp_server = "smtp.qq.com"
    server = smtplib.SMTP(smtp_server,25)
    server.login(from_addr,pwd)
    server.sendmail(from_addr,to_addr,msg.as_string()) # as_string（）把MIMEText变成一个str
    server.close()

def write_xls(ret):
	workbook = xlsxwriter.Workbook(date + '.xlsx')
	format_1 = workbook.add_format({'bold': True, 'font_size': 16, 'align': 'center'})  #添加样式1，格式为加粗、字体16号、居中
	format_2 = workbook.add_format({'font_size': 14, 'align': 'center'})   #添加样式2，字体14号、居中
	format_3 = workbook.add_format({'font_size': 14, 'align': 'center', 'num_format': 'yyyy-mm-dd hh:mm:ss'})  #添加样式3，字体14号、居中、格式化为时间日期

	worksheet = workbook.add_worksheet("sheet1")  #添加工作表
	worksheet.set_column('A:B', 30)  #将A到B列的宽度设为30
	worksheet.set_column('C:D', 40)  #将C到D列的宽度设为40

	#写入标题行 write(单元格，内容，样式)
	worksheet.write('A1', '会员手机号',format_1) 
	worksheet.write('B1','商品名称',format_1)
	worksheet.write('C1','商品单价',format_1)
	worksheet.write('D1','购买时间',format_1)

	row = 1  #从第2行开始写入
	col = 0  #从第1列开始写入
	for record in ret:  
		for s in record:

			#如果循环到第4列，将内容设置成样式3（日期格式）
			if col == 3:
				worksheet.write(row, col, s, format_3)
				continue

			#将内容写入到第row行，第col列,格式为样式2
			worksheet.write(row, col, s, format_2)
			col+=1

		row += 1  #行数加1
		col = 0   #列数归0

	workbook.close()

date = time.strftime('%Y-%m-%d',time.localtime())  #获取当前日期，用于文件命名

send_mail()  #发送邮件

三、可能遇到的问题

A、运行报错 cx_Oracle.DatabaseError: DPI-1047: Cannot locate a 64-bit Oracle Client library:

Traceback (most recent call last):
  File "C:\Users\Administrator\Desktop\movic_report.py", line 78, in <module>
    connection=cx_Oracle.connect(username, userpwd, dsn")
cx_Oracle.DatabaseError: DPI-1047: Cannot locate a 64-bit Oracle Client library: "The specified module could not be found".

原因：

1、oracle客户端和python 位数不一致

2、oracle客户端的bin目录没有加入系统环境变量中，导致找不到oracle相关库文件

解决方法：

1、安装相同位数的oracle客户端和python（同为32位或64位）

2、将oracle客户端的bin目录加入到环境变量（或者 将 oci.dll，oraocci11.dll和oraociei11.dll库文件拷贝到python目录下lib->site-packages里面）

B、查询结果包含中文，报下面编码错误

Traceback (most recent call last):
  File "E:\oracle.py", line 33, in <module>
    cursor.execute(sql)
UnicodeEncodeError: 'ascii' codec can't encode characters in position 35-39: ordinal not in range(128)

解决方法：

查看系统编码：
import sys  
print(sys.getdefaultencoding())  #输出utf8
 
查看oracle数据库编码：
select userenv('language') from dual; #结果为CHINESE_CHINA.AL32UTF8
 
方法一：
手机设置系统环境变量NLS_LANG的值：
windows下：右键计算机属性->高级系统设置->高级->环境变量->在系统变量下新建变量名：NLS_LANG，值为CHINESE_CHINA.AL32UTF8
Linux下：在/etc/profile文件末尾增加一行，export NLS_LANG=CHINESE_CHINA.AL32UTF8
 
方法二：
conn = cx_Oracle.connect(connectString, encoding = "UTF-8", nencoding = "UTF-8")

C、查询的sql语句中，如果别名使用单引号，会报错，提示未找到要求的 FROM 关键字，但是where语句中，条件的值又只能使用单引号，不然会报错，提示 标识符无效

sql = “SELECT USERID AS '列名' FROM EB_LOG WHERE USERID = 'admin' ”  #别名使用单引号，报错
 
Traceback (most recent call last):
  File "E:\oracle.py", line 34, in <module>
    cursor.execute(sql)
cx_Oracle.DatabaseError: ORA-00923: 未找到要求的 FROM 关键字
 
 
sql = 'SELECT USERID AS "列名" FROM EB_LOG WHERE USERID = "admin"'  #where条件的值使用双引号，报错
 
Traceback (most recent call last):
  File "E:\oracle.py", line 33, in <module>
    cursor.execute(sql)
cx_Oracle.DatabaseError: ORA-00904: "U": 标识符无效

解决方法：

#sql语句整体使用单引号，别名使用双引号，where条件的值使用单引号，并用反斜杠转义 
sql = 'SELECT EB_USERID AS "列名" from EB_LOG WHERE EB_USERID = \'admin\' '
