使用docker快速搭建L2TP服务器

Posted by on Sun, Apr 4, 2021

使用 Docker 镜像快速搭建 IPsec VPN 服务器。支持 IPsec/L2TP，Cisco IPsec 和 IKEv2 协议。

作者github地址：https://github.com/hwdsl2/docker-ipsec-vpn-server/blob/master/README-zh.md

 

一、下载镜像并运行

docker run \ –name ipsec-vpn-server \ –restart=always \ -v ikev2-vpn-data:/etc/ipsec.d \ -v /lib/modules:/lib/modules:ro \ -p 500:500/udp \ -p 4500:4500/udp \ -d –privileged \ hwdsl2/ipsec-vpn-server

二、获取 VPN 登录信息

docker logs ipsec-vpn-server

在命令输出中查找下面这些行：

Server IP: 你的VPN服务器IP IPsec PSK: 你的IPsec预共享密钥 Username: 你的VPN用户名 Password: 你的VPN密码

三、配置 IPsec/L2TP VPN 客户端

https://github.com/hwdsl2/setup-ipsec-vpn/blob/master/docs/clients-zh.md

四、取消所有流量走VPN

1、禁止PPTP、L2TP 修改本机缺省路由

点击桌面右下角网络图标，右键点击VPN，选择属性，点击网络-》属性-》高级-》取消“在远程网络上使用默认网关”

设置后重新连接VPN

2、设置静态路由，用于访问VPN内网

route add 192.168.100.0 mask 255.255.255.0 192.168.42.1
