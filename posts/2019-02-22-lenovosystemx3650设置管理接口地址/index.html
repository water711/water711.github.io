Lenovo System x3650 设置管理接口地址

Posted by on Fri, Feb 22, 2019

1、开启服务器。

2、显示 Setup提示后，按 F1。（此提示在屏幕上仅显示几秒钟。必须迅速按 F1。）

如果同时设置了开机密码和管理员密码，则必须输入管理员密码才能访问完整的 Setup Utility 菜单。

3、在 Setup Utility 主菜单中，选择 System Settings。

4、在下一屏幕中，选择 Integrated Management Module。

5、在下一屏幕中，选择 Network Configuration。

6、向下移动到DHCP Control，选择Static IP。

7、输入管理接口的IP地址、掩码和网关

8、向下移动到Save Network Settings

9、退出Setup utility。
