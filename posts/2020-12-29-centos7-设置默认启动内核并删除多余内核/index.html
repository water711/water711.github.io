centos7 设置默认启动内核，并删除多余内核

Dec 29, 2020
1 min read

 

设置默认启动内核

1、查看系统可用内核

[root@NAS]# awk -F\’ ‘$1==“menuentry " {print i++ " : " $2}’ /etc/grub2.cfg

0 : CentOS Linux (4.4.248-1.el7.elrepo.x86_64) 7 (Core) 1 : CentOS Linux (3.10.0-1160.11.1.el7.x86_64) 7 (Core) 2 : CentOS Linux (3.10.0-957.el7.x86_64) 7 (Core) 3 : CentOS Linux (0-rescue-70b8558dd1bf4abb887cf26344a88da3) 7 (Core) Linux (3.10.0-1127.8.2.el7.x86_64) 7 (Core)’ –class centos –class gnu-linux –class gnu –class os –unrestricted $menuentry_id_option ‘gnulinux-3.10.0-957.el7.x86_64-advanced-31cdfa8e-4bda-4e29-a11e-8e46000798d6’ {

2、查看系统开机默认启动内核

[root@NAS ~]# grub2-editenv list saved_entry=CentOS Linux (3.10.0-1160.11.1.el7.x86_64) 7 (Core)

3、查看系统当前内核

[root@NAS ~]# uname -r

4、设置默认启动内核

[root@NAS ~]# grub2-set-default ‘CentOS Linux (4.4.248-1.el7.elrepo.x86_64) 7 (Core)’

5、验证

[root@NAS ~]# grub2-editenv list

 

删除多余内核

1、查看系统安装了哪些内核包

[root@NAS ~]# rpm -qa |grep kernel

2、删除旧的内核

[root@NAS ~]# yum remove 包名
