windows下使用pip安装包失败，提示Could not install packages due to an EnvironmentError: [Errno 0] Error

Posted by on Mon, Nov 4, 2019

python27版本，使用pip安装包，报错 Could not install packages due to an EnvironmentError: [Errno 0]

重装python后，并将pip更新到最新版本，还是报错

弄了一晚上，结果发现是之前修改了cmd命令行窗口的默认编码，导致安装包失败
