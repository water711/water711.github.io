Mysql临时开启查询日志

Posted by on Fri, Jan 15, 2021

查询当前是否开启了查询日志

mysql> show global variables like ‘%general_log%’; +——————+————————————————+ | Variable_name | Value | +——————+————————————————+ | general_log | OFF | | general_log_file | D:\mysql\data\LENOVO-LA0X1198.log | +——————+————————————————+

开启查询日志

mysql> SET GLOBAL general_log = ‘ON’;

关闭查询日志

mysql> SET GLOBAL general_log = ‘OFF’;
