Python+Selenium 使用cookie实现自动登录网站

Posted by on Sun, Mar 29, 2020

Python版本：3.7

浏览器：Chrome 79.0.3

 

一、下载Selenium浏览器驱动（WebDriver）

下载地址：http://npm.taobao.org/mirrors/chromedriver/

（下载与浏览器版本对应的驱动，并将驱动放到python程序所在目录）

 

二、获取网站cookie

1、chrome浏览器安装第三方扩展EditThisCookie

2、手动输入账号密码登录网站，使用EditThisCookie导出cookies

3、将cookie（列表）转换成json文件

# -*- coding : utf-8 -*- import json

cookies = [ { “domain”: “.51cto.com”, “name”: “gr_user_id”, “value”: “ea0dbdab-1233-4454-a237-1ba0bd7c406e”, “id”: 1 }, { “domain”: “.51cto.com”, “name”: “Hm_lpvt_110fc9b2e1cae4d110b7959ee4f27e3b”, “value”: “168979366982”, “id”: 2 }, …… ]

with open(“cookie.json”, “w”) as f: json.dump(cookies,f) print(“已保存为json文件…”)

 

三、使用cookie自动登录网站

from selenium import webdriver import json

#打开浏览器 driver = webdriver.Chrome()

#访问网址 driver.get(“https://www.51cto.com/")

#最大化浏览器窗口 driver.maximize_window()

#从json文件中载入cookie,并转换回列表 with open(‘cookie.json’, ‘r’) as f: cookies = json.load(f)

#添加cookies到当前浏览器 for cookie in cookies: driver.add_cookie(cookie)

#刷新当前网页 driver.refresh()
