python 列表生成式

Posted by on Sun, Mar 26, 2017

需求：把一个列表中每个元素的值加1

第一种方法（for…in循环 + append到新列表）

a = [5, 8, 11, 14, 17, 20, 23, 26] b = [] for i in a: b.append(i+1) a = b print(a)

第二种方法（使用for…in循环 + enumerate函数）

a = [5, 8, 11, 14, 17, 20, 23, 26] for i,j in enumerate(a): a[i] = j+1 # 或者 a[i]+=1 print(a)

第三种方法(使用map函数 + lambda表达式)

a = [5, 8, 11, 14, 17, 20, 23, 26] a = map(lambda x:x+1,a) print(list(a))

第四种方法(使用列表生成式)

a = [5, 8, 11, 14, 17, 20, 23, 26] a = [i+1 for i in a] print(a)
