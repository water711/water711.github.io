Python3实现按任意键继续或退出

Oct 6, 2019
1 min read

在Windows下，我们可以使用内建的msvcrt库，实现按任意键继续或退出

一、获取按键对应的ASCII 值

import msvcrt print(ord(msvcrt.getch()))

svcrt 中的 getch() 方法，作用是获取一个按键响应并返回对应的字符。

ord 将获得的字符转换为 ASCII 数值，例如捕获按键“q”将得到数值113，按键“Q”将得到数值81

 

二、利用while死循环，当msvcrt捕获到指定的按键，中断循环

import msvcrt

print(“Press ‘q’ to exit…”)

while True: if ord(msvcrt.getch()) in [81, 113]: break
