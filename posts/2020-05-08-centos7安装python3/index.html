Centos7源码安装python3

May 8, 2020
1 min read

1、下载源码

wget https://www.python.org/ftp/python/3.8.2/Python-3.8.2.tgz tar xf Python-3.8.2.tgz cd Python-3.8.2/

2、安装编译所需依赖

yum install -y gcc openssl-devel zlib-devel libffi-devel sqlite-devel

3、编译

./configure prefix=/usr/local/python3 make -j8 && make install

#如果gcc版本>=8.1.0，在编译时可以添加优化选项–enable-optimizations ./configure prefix=/usr/local/python3 –enable-optimizations make -j8 && make install

4、将python的bin目录添加到环境变量

export PATH=$PATH:/usr/local/python3/bin/ echo export PATH=$PATH:/usr/local/python3/bin/ » /etc/profile
