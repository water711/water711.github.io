Centos7部署DNS服务

Posted by on Fri, Jun 5, 2020

 

1、安装bind软件包

[root@DNS ~]# yum -y install bind

2、修改主配置文件，添加正向解析区域

[root@DNS ~]# vim /etc/named.conf

options { listen-on port 53 { any; }; //将127.0.0.1修改为any listen-on-v6 port 53 { any; }; //将::1修改为any directory “/var/named”; //区域数据文件默认位置，可以不用修改 … allow-query { any; }; //将127.0.0.1修改为any

#在最后面添加正向解析区域 zone “test.com” { type master; //类型为主要区域 file “test.com.zone”; //区域数据文件名 };

3、新建区域数据文件

[root@DNS ~]# vim /var/named/test.com.zone @ IN SOA dns root ( 2020060500 1H 15M 1W 1D ) @ IN NS dns dns IN A 192.168.1.156 www IN A 192.168.1.156

4、启动服务

[root@DNS ~]# systemctl restart named [root@DNS ~]# systemctl enable named [root@DNS ~]# ss -lanp | grep :53

5、防火墙开放DNS服务

[root@DNS ~]# firewall-cmd –permanent –add-service=dns [root@DNS ~]# firewall-cmd –reload
