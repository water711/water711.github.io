CentOS7使用SpeedTest工具测速

Jan 13, 2021
1 min read

SpeedTest官方文档：https://www.speedtest.net/apps/cli

 

方法一：

wget -O speedtest-cli https://raw.githubusercontent.com/sivel/speedtest-cli/master/speedtest.py chmod +x speedtest-cli ./speedtest-cli –share

方法二：

sudo yum install wget wget https://bintray.com/ookla/rhel/rpm -O bintray-ookla-rhel.repo sudo mv bintray-ookla-rhel.repo /etc/yum.repos.d/ sudo yum install speedtest
