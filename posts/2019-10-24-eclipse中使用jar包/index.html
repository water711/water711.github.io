Eclipse中使用jar包

Posted by on Thu, Oct 24, 2019

一、什么是jar包

jar包就是别人已经写好的一些类，然后将这些类进行打包，你可以将这些jar包引入你的项目中，然后就可以直接使用这些jar包中的类和属性以及方法。

 

二、如何下载jar包

  • 包所在的官网下载
  • 百度或谷歌，搜索包名
  • 第三方jar包仓库下载（例如：https://mvnrepository.com/）

 

三、Eclipse如何使用jar包

1、在项目下，新建lib目录，把jar包拷贝到该目录

2、选择项目，右键点击jar包 -> Build Path -> Add to Build Path
