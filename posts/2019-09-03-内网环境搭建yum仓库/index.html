内网环境搭建YUM仓库

Sep 3, 2019
1 min read

需求：内网环境下，无法访问外网，如何更新系统以及安装项目所需软件？

一、找一台可以联网的机器，安装操作系统（一般最小化安装，和内网机器一致）缓存所需软件包

1、开启yum缓存

[root@host1 ~]# vim /etc/yum.conf [main] cachedir=/var/cache/yum/$basearch/$releasever //此处为缓存路径 keepcache=1 //设为1，则开启yum缓存 [root@host1 ~]# yum clean all

2、更新系统（缓存更新包）

[root@host1 ~]# yum -y update

3、缓存制作yum仓库所需软件

[root@host1 ~]# yum -y install createrepo vsftpd

4、缓存项目所需要的软件包

[root@host1 ~]#  yum -y install ncurses ncurses-devel openssl-devel bison gcc gcc-c++ make

5、插入U盘或移动硬盘，把yum缓存下来的软件包，拷贝出来

[root@host1 ~]# dmesg   //查看U盘设备名称 [root@host1 ~]# mount /dev/sdb /mnt/usb   //挂载 [root@host1 ~]# mkdir /mnt/usb/repo [root@host1 ~]# find /var/cache/yum/ -name “*.rpm” -exec cp -rf {} /mnt/usb/repo \;   //搜索rpm包并复制到U盘

二、内网环境下，部署yum仓库

1、安装vsftpd，把软件包拷贝到/var/ftp目录下

[root@repo ~]# rpm -ivh vsftpd-3.0.2-25.el7.x86_64.rpm [root@repo ~]# cp -rf /mnt/usb/repo /var/ftp/ [root@repo ~]# systemctl start vsftpd [root@repo ~]# systemctl enable vsftpd

2、安装repodata，并创建repodata索引文件

[root@repo ~]# rpm -ivh createrepo-0.9.9-28.el7.noarch.rpm [root@repo ~]# createrepo /var/ftp/repo

三、内网环境，找一台机器测试

1、备份原有仓库文件，配置私有yum仓库

[root@test ~]# mv /etc/yum.repos.d/{*.repo,bak/}   //备份repo文件 [root@test ~]# vi /etc/yum.repos.d/private.repo [myrepo] name=private_repo baseurl=ftp://192.168.1.90 gpgcheck=0 enable=1 [root@test ~]# yum clean all [root@test ~]# yum repolist

2、更新系统并安装软件

[root@test ~]#  yum -y update [root@test ~]#  yum -y install ncurses ncurses-devel openssl-devel bison gcc gcc-c++ make
