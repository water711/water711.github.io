python datetime模块

Jul 27, 2020
1 min read

datetime是Python处理日期和时间的标准库。

官方文档：https://docs.python.org/3/library/datetime.html

 

from datetime import datetime,timedelta

#获取当前日期时间 now = datetime.now()

#str转换为datetime mday = datetime.strptime(‘2020-01-01 12:10:55’, ‘%Y-%m-%d %H:%M:%S’) print(mday) # 2020-01-01 12:10:55 print(type(mday)) # <class ‘datetime.datetime’>

#datetime转换为str(将字符串格式化为datetime) s = mday.strftime(’%Y-%m-%d %H:%M:%S’) print(s) # 2020-01-01 12:10:55 print(type(s)) # <class ‘str’>

#datetime加减(需要导入timedelta类) day1 = mday + timedelta(days=2) #2020-01-03 12:10:55 day2 = mday + timedelta(hours=5) #2020-01-01 17:10:55 day3 = mday - timedelta(days=5, hours=6) #2019-12-27 06:10:55

 

#打印一年的日期 start_date = datetime.strptime(‘2019-01-01’,’%Y-%m-%d’) for i in range(0,365): ndate = start_date + timedelta(days=i) ndate = str(ndate).split(" “)[0] #将日期和时间切割出来，取日期 print(ndate)
